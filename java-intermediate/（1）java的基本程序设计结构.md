# 一、字符串
在标准java类库中提供了一个预定义类，叫String；每个用双引号括起来的字符串都是String类的一个实例：
```java
String e = "";//一个空的字符串
String greeting = "Hello";
```
## 子串
String类的substring方法可以从一个较大的字符串提取出一个子串。
```java
String greeting="Hello";
String s = greeting.substring(0,3);//创建了一个由字符"Hel"组成的字符串。
```
